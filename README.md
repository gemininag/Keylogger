# Java Keyboard Event Logger
---
## Overview
- Captures global key events using [JNativeHook](https://github.com/kwhat/jnativehook)  
- Saves keystrokes to `keys.txt` and logs program activity in `all.log`  
- Intended for learning **Java programming, event handling, and system input monitoring**
